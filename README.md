# 盒马 & 叮咚 运力监控脚本
# 环境：
## python 3.0 
## node.js 稳定版

该工具是通过post方式监控叮咚的运力以及用get方式监控盒马的运力。
该脚本已经解决叮咚nars生成问题。

# 盒马：
只需要在手机端打开商品页，建议挑选一个一直有货的商品推荐贵的酒类可以长期监控。

# 叮咚：
## 4月23日已更新
1. 需要抓包，手机或者电脑登录叮咚小程序。
2. 打开小程序中我的，点击收货地址。然后需要通过抓包获取 https://sunquan.api.ddxq.mobi/api/v1/user/address/ 中的信息。
3. 按抓包填入yunliall.py后执行
